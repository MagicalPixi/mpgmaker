## mpgmaker
_a command line tool for generate a [pixi](http://www.pixijs.com/) game_

##### Install

###### From npm:

`npm install -g mpgmaker`

###### Form local:

Clone the project form github: </br>
`git clone
https://github.com/MagicalPixi/mpgmaker.git`

Enter the project</br>
`cd project`

Link the command</br>
`npm link`
